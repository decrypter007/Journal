Wipe All Data?
This will permanently delete all trades, accounts, and strategies from this device. This cannot be undone.
💡 This uses a Google Apps Script deployed as a Web App — a free Google service that writes to your sheet. No API key or OAuth needed.
Step 1 — Create your Google Sheet
Go to sheets.google.com, create a new spreadsheet. The name of the first tab doesn't matter — the script uses whatever your sheet has.
Step 2 — Open Apps Script
In your sheet, click Extensions → Apps Script. A new script editor opens. Delete any existing code and paste in this script exactly:
function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  if (data.action === 'writeMulti') {
    var tabMap = {trades:'Trades',strategies:'Strategies',accounts:'Accounts',transactions:'Transactions'};
    for (var key in tabMap) {
      var rows = data.sheets[key];
      if (!rows || !rows.length) continue;
      var sh = ss.getSheetByName(tabMap[key]) || ss.insertSheet(tabMap[key]);
      sh.clearContents();
      sh.getRange(1,1,rows.length,rows[0].length).setValues(rows);
    }
  } else {
    var sh = ss.getActiveSheet();
    sh.clearContents();
    sh.getRange(1,1,data.rows.length,data.rows[0].length).setValues(data.rows);
  }
  return ContentService.createTextOutput(JSON.stringify({status:'ok'})).setMimeType(ContentService.MimeType.JSON);
}
Step 3 — Deploy as Web App
1. Click Deploy → New deployment
2. Click the gear ⚙ next to Type → select Web App
3. Set Execute as: Me
4. Set Who has access: Anyone
5. Click Deploy → authorize when prompted
6. Copy the Web App URL (looks like https://script.google.com/macros/s/…/exec)
Step 4 — Paste URL in APEX
Switch to the Setup tab, paste the Web App URL, then tap Save & Test Connection. Your trades will be written to the sheet immediately.
Updating the script later
If you ever change the Apps Script code, you must create a New Deployment (not update existing) to get a new URL. Old URLs keep running the old version.
✅ Once connected, APEX syncs automatically 3 seconds after every trade save. Use Sync Now in settings to force an immediate sync.
